# Tenant Admin Dashboard · Remaining Roadmap

This document tracks the outstanding phases (4–10) for the Tenant Admin Dashboard effort.

## Phase 4 – Departments, Roles, Approvals
- [ ] **Department management UI**: support global vs branch-scoped departments, shared departments, and per-module enablement toggles.
- [ ] **Role builder**: surface system/tenant/region/branch roles, expose custom permission sets, and render an RBAC matrix.
- [ ] **Approval flow designer**: provide a multi-level approval editor with regional escalation paths, backup approvers, and SLA targets.

## Phase 5 – Users & Employees
- [ ] **Employee assignment console**: map employees across subsidiary → region → country → branch → department with multi-location support.
- [ ] **Access control panel**: configure role/module/location-based access plus temporary access grants.
- [ ] **Lifecycle workflows**: orchestrate onboarding, transfer, promotion, and exit checklists.

## Phase 6 – Modules & Feature Flags
- [ ] **Module registry**: enable/disable modules per subsidiary/region, with visibility into status and dependencies.
- [ ] **Feature flag controls**: manage beta toggles, region-specific features, and compliance-restricted flags.

## Phase 7 – Billing & Subscription
- [ ] **Subscription overview widget**: show plan details, module activations, headcount usage vs limits, and upcoming billing.
- [ ] **Cost allocation dashboards**: deliver chargeback reporting per subsidiary/region/branch with internal summaries.
- [ ] **Invoices & payments**: list invoices, support downloads, display payment history, and proration views.

## Phase 8 – Integrations & APIs
- [ ] **Integrations manager**: configure payment/email/LMS/comms stacks with status indicators and settings.
- [ ] **API & webhook center**: issue keys per environment, define region-specific webhooks, track rate limits, and expose logs.

## Phase 9 – Analytics & Reporting
- [ ] **Global analytics suite**: present workforce, financial, productivity, and project metrics with advanced filters.
- [ ] **Export & scheduling tooling**: deliver PDF/Excel/CSV exports plus scheduled deliveries.

## Phase 10 – Security & Compliance
- [ ] **Security console**: enforce MFA, apply IP restrictions, and monitor sessions/devices.
- [ ] **Audit trails**: capture user & role changes, approvals, and sensitive data access logs.
- [ ] **Compliance settings**: manage GDPR/NITDA/SOC controls and map data residency.
